MorphoFace Installation

Pre-Requisites

1. Windows 10
2. Windows partitions C and D
3. User accounts
	MFACE-DISPLAY/MorphoMFACE (User)
	MFACE/MorphoMFACE3$ (Admin)
	MFACE-SERVICE/MorphoMFACE3$ (Admin, created by MFACE installer)
4. Auto-login MFACE-DISPLAY
	Disable MFace Display Service auto-start
		Open 'Task Manager'
		Select the Startup tab
		Right click on the MFace.Display entry and select Disable
	Automatically log in as MFACE-DISPLAY
		Run 'netplwiz'
		Select the user to login automatically
		Unselect "Users must enter a user name and password to use this computer", click Apply
		Enter the user's password, click OK
5. Automatically start up the MorphoFace application
		Copy D:\MorphoFace\scripts\MorphoFace.lnk to C:\Users\MFACE-DISPLAY\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

		
Required Software

1. Install Morpho Multiprotect
	Run the standard Morpho License Manager installer
	
2. MFACE SDK and Service
	Run the mface-2.x.x-setup.exe installer
	
3. Java 8 JRE (latest)
	Run the Java Runtime Environment x64 installer
	
3. MorphoFace Application Software
	Extract MorphoFace-2.0.x.zip to D:\MorphoFace

	
Optional Software

- MFace Tools
	Unzip mface-2.x.x-tools.zip to new directory D:\MFace.Tools
- 7Zip
- Notepad++
- WinMerge


MFACE Configuration

1. Generate and install Morpho licenses
	MIMA, MORPHOFACS, VERIF
	Restart the Sagem License Service

2. Disable MFACE auto-reboot on crash
	Open Services
	Right click on MFACE Service > Properties
	Select Recovery tab
	Change "Restart the Computer" to "Restart the Service" for Subsequent failures
	Click OK
	
3. Apply the MFACE configuration files
	Back up D:\MFace.Display\MFace.Display.exe.config to D:\MFace.Display\MFace.Display.exe.config.orig
	Copy D:\MorphoFace\conf\MFace\MFace.Display\MFace.Display.exe.config to D:\MFace.Display\MFace.Display.exe.config

	Back up D:\MFace.Service\configMW.XML to D:\MFace.Service\configMW.XML.orig
	Copy D:\MorphoFace\conf\MFace\MFace.Service\configMW.XML to D:\MFace.Service\configMW.XML
	Restart the MFACE Service in the Services control panel
